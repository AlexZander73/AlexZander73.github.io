Support

Help for setup, widgets, reminders, and data export.

Common fixes

  • If widgets look stale, open Capdrop once so iOS refreshes the local widget snapshot.
  • If notifications do not appear, check iOS Settings and confirm notification permission is enabled.
  • If HealthKit writes fail, confirm Apple Health permission for water data is enabled.
  • If alternate icons do not change, confirm the installed build includes the alternate icon assets.

Report an issue

Use the public repository issue tracker for launch support: github.com/AlexZander73/DrinkWaterApp/issues.

Data reset

Capdrop stores data locally. To fully reset, use the app settings export/reset flow if available, or remove the app from the device.
