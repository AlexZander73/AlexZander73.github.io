What it is

Listen Library is a multi-platform storefront and personal library system for ebooks and audiobooks. It pairs discovery with organization: collections, tags, and smart rules that keep your shelves fresh as you read and listen.

Why it exists

Digital libraries spread across apps and devices. Listen Library keeps purchases and imports together, with clear progress tracking and a calm, premium browsing flow.

For who

Readers and listeners who want structure, migration support, and a clean library OS that does not feel cluttered.

FAQ

Is this live in app stores?

Not yet. The repository contains the web app, mobile app, and API setup.

Does it handle DRM?

The current focus is DRM-free delivery, with a license abstraction for future DRM.

Can I import from other services?

Yes, via metadata-only CSV import and DRM-free file upload.
